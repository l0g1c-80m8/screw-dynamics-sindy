# Analysis Notebooks (`notebook/`)

Jupyter notebooks for data analysis, model evaluation, and visualization of SINDy modeling results.

## Files

### Core Analysis
- **`final-sindy-results.ipynb`**: Main results analysis and model comparison
- **`plot_final.ipynb`**: Final result visualization and publication plots
- **`simulation-time-results.ipynb`**: Temporal analysis and dynamics evaluation

### Data Processing
- **`post_process.ipynb`**: Data preprocessing and feature extraction
- **`refine_data.ipynb`**: Data cleaning and quality assessment
- **`extract_best_signal.ipynb`**: Signal processing and noise reduction
- **`smooth_raw_pixels.ipynb`**: Image-based data smoothing and filtering

### Experimental Analysis
- **`hakuna-matata.ipynb`**: Exploratory data analysis

## Quick Usage

```bash
# Start Jupyter Lab
jupyter lab notebook/

# Or run specific notebook
jupyter lab notebook/final-sindy-results.ipynb
```

## Notebook Features

- **Model comparison**: SINDy vs LSTM vs MLP performance analysis
- **Visualization**: Interactive plots and publication-ready figures
- **Data processing**: Signal extraction and noise reduction pipelines
- **Results interpretation**: Dynamics analysis and coefficient examination
